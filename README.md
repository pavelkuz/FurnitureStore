# FurnitureStore
C# project
В приложении можно:
1. Создавать роли
2. Удалять роли
3. Редактировать роли
4. Регистрировать пользователя
5. Авторизоваться
6. В зависимости от прав доступны разные функции
7. UserID передается через куки и сверяется на операции создания роли
8. Есть каталог просмотра товаров
